How to Use this Application - TLDR; Version

  1. Enter the URL for the site from which you want to get keywords.
  2. Enter the keyword for which to search. This keyword will add 'weight' to the keywords and allow them to be sorted by relevancy.
  3. Select BFS for a small site or DFS for a large site.
  4. Select search and get your keywords!
